- ** Java vervion : 1.7**
- ** Spring Boot version : '3.0.4'
- ** Javascript**
- ** Bootstrap version : 3.3.7**
- ** jQuery version :  3.1.1**
- ** Lombok version : '1.18.24'
- ** thymeleaf
- ** data-jpa
- ** h2database:h2
